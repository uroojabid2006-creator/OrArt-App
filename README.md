# 🎨 OrArt – Creative Art Marketplace & Design Studio

**OrArt** is a modern Flutter-based mobile application that seamlessly combines an **art marketplace** with a **creative design studio**, offering users a unique platform to both **discover beautiful artwork** and **create personalized designs**. The application provides a smooth and visually appealing experience while enabling efficient product and order management through an integrated admin system.

---

## ✨ Key Features

### 🛒 Art Marketplace

Explore and shop from a diverse collection of handmade and creative artworks.

* Browse categories such as **Paintings, Calligraphy, Crochet**, and more.
* View detailed product descriptions and images.
* Add items to the **Cart** and **Wishlist**.
* User-friendly interface with smooth navigation.
* Elegant pastel-inspired design for an engaging experience.

---

### 🖌️ Creative Design Studio

Create custom artwork directly within the app using built-in design tools.

* Add customizable text with multiple font styles.
* Freehand drawing with adjustable brush sizes.
* Insert stickers, shapes, and apply custom colors.
* Import images from the device gallery.
* Save and export personalized designs.

---

### 🧩 Admin Management Panel

A dedicated admin module designed for efficient marketplace management.

* Upload and manage artwork collections.
* Update product information and pricing.
* Monitor and process customer orders.
* Maintain inventory and product availability.
* Simplify marketplace operations through centralized management.

---

### 🛍️ Shopping Cart & Order System

Provide users with a convenient and secure purchasing experience.

* Add and remove products from the cart.
* Simple checkout process with customer information.
* Support for **Cash on Delivery (COD)**.
* Order tracking and management for administrators.

---

## 🎨 User Interface & Experience

OrArt is designed with creativity and simplicity in mind.

* Soft pastel color palette.
* Modern and clean interface.
* Smooth animations and transitions.
* Intuitive navigation for an enhanced user experience.
* Art-inspired visual elements that reflect creativity and elegance.

## 📱 Project Overview

**OrArt** bridges the gap between **art creation** and **e-commerce** by providing a single platform where users can:

* Discover and purchase unique artworks.
* Create personalized designs using integrated creative tools.
* Enjoy a seamless shopping experience with a visually appealing interface.

The project aims to empower creativity while making art more accessible and interactive.
